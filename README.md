**A Rainmeter-based animation in which text alternately appears and disappears with fade-in/fade-out and sliding effects.**

**基于 Rainmeter 的动画效果，文本会以淡入淡出和滑动的方式交替出现与消失。**

**Rainmeterベースのアニメーションで、テキストがフェードイン／フェードアウトとスライド効果によって交互に表示・非表示されます。**

**Rainmeter 기반 애니메이션으로, 텍스트가 페이드 인/페이드 아웃과 슬라이딩 효과를 통해 번갈아 나타나고 사라집니다.**
